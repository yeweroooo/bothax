# GetWorld

Ringkasan
- `GetWorld() -> World`
- Mengambil info world yang sedang dimuat.

Nilai kembali
- `World`

Contoh (Lua)
```lua
local world = GetWorld()
LogToConsole('you are in world : ' .. world.name)
LogToConsole('world size : ' .. world.width .. ' x ' .. world.height)
```

Terkait
- `World`
