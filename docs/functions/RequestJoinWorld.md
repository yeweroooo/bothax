# RequestJoinWorld(worldname: string)

Ringkasan
- `RequestJoinWorld(worldname: string)`
- Meminta pindah ke world tertentu (setara perintah `/join`/`/wp`).

Penjelasan
- Men-trigger proses perpindahan dunia seperti mengetik perintah di chat. Pastikan nama world valid.
