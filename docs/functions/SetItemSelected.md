# SetItemSelected(itemid: int)

Ringkasan
- `SetItemSelected(itemid: int)`
- Memilih item dari inventaris berdasarkan ID-nya.

Penjelasan
- Meniru aksi memilih item seperti di UI permainan. Pastikan `itemid` ada di inventaris.

Nilai kembali
